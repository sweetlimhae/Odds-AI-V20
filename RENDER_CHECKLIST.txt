Render 배포 체크리스트

1. GitHub 저장소 루트에 app.py가 있어야 함
   - app.py
   - requirements.txt
   - templates/index.html
   - static/app.js
   - static/style.css

2. Render 설정
   Build Command:
   pip install -r requirements.txt

   Start Command:
   gunicorn app:app

3. 배포 후 확인 주소
   https://본인주소.onrender.com/
   https://본인주소.onrender.com/health

4. Not Found가 뜨면
   - 예전 서비스 URL을 보고 있는지 확인
   - GitHub에 새 파일이 Push 되었는지 확인
   - Render Manual Deploy > Clear build cache & deploy 실행
   - Root Directory가 비어 있어야 함. app.py가 하위 폴더에 있으면 그 폴더명을 Root Directory에 입력
